## Codebase Patterns

- External canonical CLI entrypoint is `littlecrab`, implemented by `opencrab.cli:main`.
- MCP stdio and CLI share the same local runtime.
- Current live store ownership is distributed:
  - Ladybug for ontology graph truth
  - DuckDB for documentary/provenance/registry/policy truth
  - Chroma for derived retrieval indexing
- `opencrab.mcp.tools` currently assembles engines directly from cached stores.
- `ontology_query` is the best first read path to route through a single context pipeline.
- Scoped filters already intentionally suppress graph expansion and must keep doing so.
